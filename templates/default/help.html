Help & Documentation

Below is a breif list of instructions that you can use as a guide for this app. To access more documentation, please visit the Lira Cash website at: http://www.lira.cash/documentations

  • Creating & Importing Wallet

    You can use LiraPay to create or import existing ethereum based wallets.

    You can create and import a wallet after installation. Additonal wallets can also be added in the Settings menu by clicking Add New Wallet.

  • Send & Receive funds

    To send funds click Send on the Dashboard menu, following the instructions and input the required details. Each transaction will require authentication (by entering PIN) and a variable gas fee in Gwei. To understand how the gas fee works, read the official Ethereum documentation here.

    Other users can send LIRA to your account. You can send your wallet address to other users by clicking Share. Other users can also scan your wallet address' QR code from the Receive Funds page accessible from the Dashboard menu.

  • Payments using LiraPay

    The LiraPay mobile application will be designed to work with the LiraPay Pro (for Merchants). Using the LiraPay mobile app you can quickly pay for goods and services by scanning the invoice QR code provided to you by your merchant in their invoices.

    To do this, simply click Send from the Dashboard menu and click Pay with Lira in the following page and proceed to complete your payment.

  • Transaction Logs

    All transactions are archived in the Ethereum network. Your transaction logs can accessed directly from the LiraPay app or in any Ethereum Transaction explorers such as Etherscan (https://etherscan.io).

    To view a transaction detail, click on a transaction from the Dashboard or History menu to display the time, amount of funds sent/received, hash, payee, and gas cost (in Gwei).

  • Wallet Security

    Your wallet data is secured using industry standard encryption in your mobile device. Your PIN encrypts your wallet private key stored in the wallets.json file included with the app. It is important to note that when disposing of the app (uninstalling) the data must also be erased if not erased automatically by the system. This can be done by accessing your mobile device's storage main folder and removing the wallet data file called wallets.json.
